# Web Standards

Daily web platform news.

## Content

News body text is a single paragraph in Markdown format, with only inline code blocks allowed. Basically, plain text with backticks for code snippets.

The total length, combining title, body, and the link (counted as 23 characters), must not exceed 500 characters.

## Covers

- Create a 1920 × 1080 cover and export it as `cover.png`.
- Convert it to AVIF via `avifenc -q 50 --speed 5 cover.png cover.avif`.
- Use the AVIF version as the source file.
