CSS gap decorations let you style gaps in grid and flexbox layouts. The `shape-outside` property now supports `path()`, `shape()`, `rect()`, and `xywh()` functions. SVG `path-length` becomes a CSS property. Text overflow switches from `ellipsis` to `clip` on user interaction. Other additions include `image-rendering: crisp-edges`, `Intl.Locale.prototype.variants`, and WebSocket connections that no longer block back/forward cache.
