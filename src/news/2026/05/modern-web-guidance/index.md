A set of evergreen, expert-vetted skills for AI coding agents from the Chrome team, covering UI components, legacy code modernization, security, and performance. Install via `npx modern-web-guidance@latest install`, or as plugins for Claude Code, Copilot CLI, Vercel Agent Skills, and Antigravity. The skills can help with modern features like `<dialog>` modals, popover tooltips, container queries, passkeys, CSP, and improving Web Vitals.
