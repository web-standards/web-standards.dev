`AccentColor` and `AccentColorText` expose the system accent color to installed apps. The `polygon()` shape function gains optional corner rounding, and `zoom` becomes animatable. CSS `url()` now accepts `cross-origin()`, `integrity()`, and `referrer-policy()` modifiers. The new `text-fit` property scales font size to fill its box, `background-clip: border-area` enables gradient borders, and scroll methods return promises.
