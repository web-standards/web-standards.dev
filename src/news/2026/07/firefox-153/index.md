This release adds `Intl.Locale` info methods like `getCalendars()`, `getWeekInfo()`, and `getTextInfo()`, the `Error.stackTraceLimit` property, `import source` for loading unevaluated modules, `getAllRecords()` for IndexedDB object stores and indexes, desktop Picture-in-Picture API support, WebAssembly JS Promise Integration via `WebAssembly.Suspending`, and the experimental `link-parameters` CSS property for passing parameters to SVG.
